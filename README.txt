README створений для невеликого опису, щоб не писати коментарі в обох файлах.
Histogram - створена для гістограми по масивам чисел, Histo - більше по умові, тобто без конкретних чисел з приблизними значеннями.
Histogram {
m - кількість стобчиків
numbers - саме ці значення, які зберігаються
min_hist - найменший елемент гістограми
max_hist - найбільший
frequency - значення відповідних стобчиків
borders - числа що відділяють значення одного стовбчика від іншого, тобто межі
}


Histo {
m - кількість стобчиків
min_hist - найменший елемент гістограми
max_hist - найбільший
frequency - frequency - значення відповідних стобчиків
values -  приблизні значення гістограми
}

Всі функції описані в завданні оформлені як там і написано, інші:

double err - максимальна похибка
double range - розмах
double dev - відхилення
void addFromFile - додати в Histo з файлу значення (числа в одному рядку через пробіли)
Histogram(String filename) - формат файлу (в першому рядку кількість рядків, в другому - числа-значення через пробіли)

Створені для реалізації функцій:

void addX - додати елемент в масив
void reassign - обрахувати в Histogram всі параметри з нових значень
void revalue - обрахувати нові значення в Histo з параметрів
